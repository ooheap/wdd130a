Project Reflection

The most challenging part of the project was learning to use GitHub. I have always been a little confused on how to use GitHub but I overcame the problems I had. I enjoyed designing my own webpage with a site plan and wireframe and seeing it come to life. If I were to do this project again I would get in contact with my junior devs much earlier and collaborate with them much more. I feel confident working on real-world projects after this experience because I feel confident in using GitHub and communicating with a team to distribute a workload.
